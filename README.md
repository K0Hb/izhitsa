# Ижица

Перевод современного русского текста на дореволюционную орфографию

## Установка

```ruby
# Gemfile
gem "izhitsa"
```
А затем выполнить:

    $ bundle install

Или установите его самостоятельно как:

    $ gem install izhitsa

### Поддерживаемые Ruby версии

- Ruby (MRI) >= 2.7.0
- JRuby >= 9.3.0
